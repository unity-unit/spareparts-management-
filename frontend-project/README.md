# PSSMS Frontend

## Overview
This React frontend uses Tailwind CSS to provide responsive forms and reports for the Parking Sales Management System.

## Setup
1. Install dependencies:
```bash
npm install
```
2. Start the app:
```bash
npm run dev
```
